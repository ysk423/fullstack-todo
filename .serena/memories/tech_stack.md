# Tech Stack

## フロントエンド (`frontend/`)
- React 19 + TypeScript 6
- Vite 8（ビルド・開発サーバー）
- テスト: Vitest + @testing-library/react + jsdom

## バックエンド (`backend/`)
- Node.js 24 + Express 4 + TypeScript 5
- DB: `node:sqlite`（Node.js 24 組み込み SQLite、外部パッケージ不要）
- テスト: Jest + supertest + ts-jest

## プラットフォーム
- OS: Windows 11 Enterprise
- Node.js: v24.16.0
- Shell: PowerShell + Bash
